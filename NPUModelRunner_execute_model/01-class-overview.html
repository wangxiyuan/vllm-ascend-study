类定义

NPUModelRunner 继承自 vLLM 的 GPUModelRunner 类，针对华为昇腾 NPU 进行了适配和优化。

class NPUModelRunner(GPUModelRunner):
    def __init__(self, vllm_config: VllmConfig, device: torch.device):
        # PCP 填充令牌数计算
        max_pcp_pad_tokens = (
            vllm_config.parallel_config.prefill_context_parallel_size
            * 2 * vllm_config.scheduler_config.max_num_seqs
        )
        vllm_config.scheduler_config.max_num_batched_tokens += max_pcp_pad_tokens

        # 调用父类初始化
        with _torch_cuda_wrapper():
            super().__init__(vllm_config, device)

Ascend 特定适配

_torch_cuda_wrapper() 上下文管理器用于临时将 CUDA 调用重定向到 NPU，确保父类初始化过程中的 CUDA 操作能够正确映射到昇腾设备。

注意力状态

AscendAttentionState 定义了不同的注意力计算状态，用于选择不同的内核实现：

class AscendAttentionState(Enum):
    PrefillNoCache = 0     # 首次 prefill，无 KV 缓存
    PrefillCacheHit = 1    # Prefill 有缓存命中
    DecodeOnly = 2         # 纯 decode 阶段
    ChunkedPrefill = 3     # 分块 prefill
    SpecDecoding = 4       # 投机解码
