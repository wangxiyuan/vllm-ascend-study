方法概述

sample_tokens 方法是推理流程的最后阶段，负责从模型输出的 logits 中采样得到下一个 token。 该方法处理采样、logprobs 计算、投机解码验证以及输出结果的组装。

_sample 方法实现

def _sample(self, logits, spec_decode_metadata):
    sampling_metadata = self.input_batch.sampling_metadata

    if spec_decode_metadata is None:
        # 普通采样
        if lmhead_tp_enable() and logits is not None:
            logits = logits[: self.input_batch.num_reqs]
        return self.sampler(
            logits=logits,
            sampling_metadata=sampling_metadata,
        )

    # 投机解码验证采样
    if lmhead_tp_enable() and logits is not None:
        logits = logits[: len(spec_decode_metadata.logits_indices)]
    sampler_output = self.rejection_sampler(
        spec_decode_metadata,
        None,  # draft_probs
        logits,
        sampling_metadata,
    )
    return sampler_output

_bookkeeping_sync 详解

该方法负责同步采样结果并更新状态：

def _bookkeeping_sync(
    self,
    scheduler_output: "SchedulerOutput",
    sampler_output: SamplerOutput,
    logits: torch.Tensor | None,
    hidden_states: torch.Tensor,
    num_scheduled_tokens: int,
    spec_decode_metadata: SpecDecodeMetadata | None,
) -> tuple[...]:
    # 处理丢弃的请求
    discard_sampled_tokens_req_indices = self.discard_request_indices.np[: self.num_discarded_requests]

    # 复制请求 ID 映射（防止异步调度时被修改）
    req_ids_output_copy = self.input_batch.req_ids.copy()
    req_id_to_index_output_copy = self.input_batch.req_id_to_index.copy()

    # 解析采样结果
    if not self.use_async_scheduling:
        if max_gen_len == 1:
            # 普通采样
            valid_sampled_token_ids = self._to_list(sampled_token_ids)
        else:
            # 投机解码采样
            valid_sampled_token_ids, cu_num_tokens = RejectionSampler.parse_output(
                sampled_token_ids,
                self.input_batch.vocab_size,
                discard_sampled_tokens_req_indices,
                logprobs_tensors=logprobs_tensors,
            )

    # 缓存采样的 tokens
    for req_idx in range(num_sampled_tokens):
        start_idx = self.input_batch.num_tokens_no_spec[req_idx]
        end_idx = start_idx + num_sampled_ids
        self.input_batch.token_ids_cpu[req_idx, start_idx:end_idx] = sampled_ids
        self.input_batch.num_tokens[req_idx] = end_idx

    return (logprobs_lists, valid_sampled_token_ids, ...)

关键点总结

  • 状态管理：execute_model_state 存储前向传播的中间结果
  • Grammar 支持：通过 bitmask 实现结构化输出

  • Bookkeeping：同步采样结果并更新请求状态
