第六章：_model_forward - 模型前向传播

vllm_ascend/worker/model_runner_v1.py:1661-1697

方法概述

_model_forward 方法是模型推理的核心执行单元，负责调用实际的模型进行前向传播。 该方法接收预处理后的输入张量，执行模型计算，并返回隐藏状态（hidden states）。

核心实现

1. 模型前向传播

def _model_forward(
    self,
    num_tokens_padded: int,
    input_ids: torch.Tensor | None = None,
    positions: torch.Tensor | None = None,
    intermediate_tensors: IntermediateTensors | None = None,
    inputs_embeds: torch.Tensor | None = None,
    **model_kwargs: dict[str, Any],
):
    assert self.model is not None

    # 调用模型进行前向传播
    hidden_states = self.model(
        input_ids=input_ids,
        positions=positions,
        intermediate_tensors=intermediate_tensors,
        inputs_embeds=inputs_embeds,
        **model_kwargs,
    )

2. ACL Graph 参数更新

ACL Graph（昇腾计算图）

ACL Graph 是昇腾 NPU 上的计算图优化技术，类似于 CUDA Graph。 在 FULL Graph 模式下运行时，需要更新图参数以确保正确执行。 

forward_context = get_forward_context()
assert forward_context is not None

if (
    forward_context.cudagraph_runtime_mode == CUDAGraphMode.FULL
    and not forward_context.capturing
    and not self.use_sparse
):
    assert positions is not None
    update_full_graph_params(
        self.attn_backend,
        self.update_stream,
        forward_context,
        num_tokens_padded,
        self.vllm_config,
        self.speculative_config,
        positions.shape[0],
    )

3. 序列并行 All-Gather

当启用 flash_comm_v1 时，需要执行 all-gather 操作收集所有 hidden states：

if get_forward_context().flash_comm_v1_enabled and not isinstance(hidden_states, IntermediateTensors):
    hidden_states = self._all_gather_hidden_states_and_aux(hidden_states)

return hidden_states

Hidden States All-Gather

单张量 All-Gather

@staticmethod
def _all_gather_hidden_states(hidden_states):
    hidden_states = tensor_model_parallel_all_gather(hidden_states, 0)
    pad_size = get_forward_context().pad_size
    if pad_size > 0:
        hidden_states = hidden_states[:-pad_size, :]
    return hidden_states

多张量 All-Gather

@staticmethod
def _all_gather_hidden_states_and_aux(hidden_states):
    if isinstance(hidden_states, tuple):
        return (
            NPUModelRunner._all_gather_hidden_states(hidden_states[0]),
            NPUModelRunner._all_gather_hidden_states_list(hidden_states[1]),
        )
    return NPUModelRunner._all_gather_hidden_states(hidden_states)

返回值

场景 返回类型 说明
非 PP last rank IntermediateTensors 中间张量，传递给下一 PP 阶段
PP last rank torch.Tensor hidden_states，用于后续 logits 计算
Pooling 模型 PoolerOutput 池化后的输出

关键点总结

  • 模型调用：通过 self.model() 执行实际的前向传播
  • ACL Graph：FULL 模式下需要更新图参数
  • 序列并行：启用 flash_comm_v1 时执行 all-gather
  • PCP 恢复：长序列并行后需要恢复 hidden states 顺序
  • Logits 计算：仅在 PP last rank 执行
