NPUWorker::get_kv_cache_spec - KV Cache 规范获取流程

概述

本文档深入分析 vLLM Ascend 中 get_kv_cache_spec 方法的完整调用链， 该方法用于获取模型中各层的 KV Cache 规范定义。KV Cache 规范描述了每个层需要什么样的 KV Cache， 包括块大小、数据类型、注意力头数等关键参数。

方法功能

get_kv_cache_spec 方法的核心职责是：

  • 遍历模型层：发现模型中所有需要 KV Cache 的层
  • 收集规范信息：从每个层获取其 KV Cache 规范
  • 构建映射表：返回层名到 KVCacheSpec 的字典映射
