拒绝采样 (Rejection Sampling)

概述

RejectionSampler 是 Speculative Decoding 的核心组件，负责根据 target model 的 logits 决定接受或拒绝草稿 token。它实现了标准的拒绝采样算法，确保输出分布与原始模型一致。

位置: vllm/v1/sample/rejection_sampler.py

RejectionSampler 工作原理

graph LR A[Draft Tokens] --> B[Target Logits] B --> C{Accept?} C -->|Yes| D[Accept Token] C -->|No| E[Resample Token] D --> F{All Accepted?} F -->|Yes| G[Sample Bonus Token] F -->|No| H[Output Accepted + Resampled] G --> I[Output + Bonus] E --> H

核心算法

┌─────────────────────────────────────────────────────────────────┐
│                    Rejection Sampling                           │
├─────────────────────────────────────────────────────────────────┤
│  Input:                                                         │
│  - draft_token_ids: [d1, d2, d3]  (draft model 生成)             │
│  - target_logits:   [t1, t2, t3, t_bonus]  (target model 计算)   │
│                                                                 │
│  Process:                                                       │
│  1. 对于每个 draft token di:                                    │
│     - 计算 acceptance probability: p = min(1, target(di)/draft(di))  │
│     - 以概率 p accept，否则 reject                              │
│                                                                 │
│  2. 如果 accept 所有 draft tokens:                              │
│     - 从 target model 采样 1 个 bonus token                      │
│     - 输出：[d1, d2, d3, bonus_token]                           │
│                                                                 │
│  3. 如果 reject 在第 k 个 token:                                │
│     - 从 adjusted distribution 采样 resampled token            │
│     - 输出：[d1, d2, ..., d(k-1), resampled_token]              │
│                                                                 │
│  Output:                                                        │
│  - sampled_token_ids: shape (batch_size, num_accepted + 1)      │
│    例如：[[d1, d2, r1], [d1', d2', d3', r1'], ...]              │
└─────────────────────────────────────────────────────────────────┘

采样流程

位置: model_runner_v1.py:1537-1556

def _sample(self, logits, spec_decode_metadata):
    sampling_metadata = self.input_batch.sampling_metadata

    if spec_decode_metadata is None:
        # 普通采样（无 spec decode）
        return self.sampler(logits=logits, sampling_metadata=sampling_metadata)

    # Spec decode 采样：验证 draft token
    if lmhead_tp_enable() and logits is not None:
        logits = logits[: len(spec_decode_metadata.logits_indices)]

    sampler_output = self.rejection_sampler(
        spec_decode_metadata,
        None,  # draft_probs
        logits,       # target model 的 logits [num_tokens + batch_size, vocab_size]
        sampling_metadata,
    )
    return sampler_output

接受概率计算

对于每个草稿 token γ，接受概率的计算公式为： 

P(accept) = min(1, P_target(γ) / P_draft(γ))

其中:
- P_target(γ) 是目标模型生成 token γ 的概率
- P_draft(γ) 是草稿模型生成 token γ 的概率

如果 P_target(γ) ≥ P_draft(γ)，则必定接受
如果 P_target(γ) < P_draft(γ)，则以概率 P_target(γ) / P_draft(γ) 接受

输出格式

RejectionSampler 的输出是一个 SamplerOutput，包含：

  • sampled_token_ids: shape (batch_size, num_accepted + 1)
  • 每个请求的输出包含接受的 draft tokens + 1 个新 token（bonus 或 resampled）
